## 개요
- 학생들이 학위 취득을 위해 필요한 졸업 신청 절차를 간편하게 수행할 수 있도록 하는 온라인 플랫폼
- 학생들의 편의를 위해 개발되었으며, 학사 관리 부서 및 학생 자체에게도 효율적인 업무 처리를 지원
- 프로젝트 개발기간: 2023.02-2023.06
